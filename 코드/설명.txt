[API 키 및 경로 설정]

데이터 수집 코드를 실행하기 위해서는 API 키 설정이 필요합니다.

1. .env 파일 생성
프로젝트 데이터수집 파일 경로에 `.env` 파일을 생성하여 API 키를 저장합니다.  
데이터 수집 코드는 해당 `.env` 파일을 읽어 API 키를 불러오도록 구성되어 있습니다.

예시
OPENAI_API_KEY=...
SUPABASE_URL=...
SUPABASE_KEY=...

2. 수집 코드 외 설정
일부 코드에서는 API 키가 하드코딩되어 있으므로, 실행 환경에 맞게 별도로 설정이 필요합니다.

3. 경로 설정
데이터 저장 및 로드를 위해 로컬 환경에 맞는 경로 설정이 필요합니다.  
예를 들어 데이터 폴더 위치를 코드 상단에서 다음과 같이 지정합니다.

DATA_DIR = "데이터 폴더 경로"

각 환경마다 폴더 위치가 다를 수 있으므로 실행 전에 경로를 확인하고 수정해야 합니다.
